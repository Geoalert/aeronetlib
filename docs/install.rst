Requirements and installation
=============================

1. python 3
2. rasterio >= 1.0.0

Pypi package:

.. code:: bash

    $ pip install aeronet

Source code:

.. code:: bash

    $ pip install git+https://github.com/aeronetlab/aeronetlib

You can also use the docker image with current version installed:

.. code:: bash

    $ docker pull aeronetlab/dev:latest
